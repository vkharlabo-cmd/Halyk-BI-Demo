# Спецификация страниц Power BI

Формат холста: 16:9. Денежные меры: тенге, разделитель тысяч, сокращение «млн» или «тыс». Проценты: один знак после запятой.

## 1. Обзор для руководства

Верхний ряд — карточки:

- Активные клиенты;
- Новые клиенты;
- Торговый оборот;
- Комиссионный доход;
- Количество сделок;
- Чистый денежный приток.

Ниже:

- линейный график комиссионного дохода по месяцам;
- отдельный линейный график торгового оборота по месяцам;
- столбчатый график чистого притока по месяцам;
- фильтры: период, сегмент клиента, подразделение, рынок.

Не помещайте оборот и комиссию на одну шкалу: значения отличаются на порядки, поэтому одна линия визуально исчезнет.

## 2. Активность клиентов

- карточка «Конверсия в первую сделку»;
- карточка «Неактивные клиенты 90 дней»;
- линии активных и новых клиентов по месяцам;
- воронка: зарегистрированные клиенты → открытые счета → клиенты с первой сделкой;
- горизонтальные столбцы клиентов по каналам привлечения;
- матрица сегмент × статус клиента.

## 3. Доход и инструменты

- горизонтальные столбцы оборота по тикерам;
- горизонтальные столбцы комиссионного дохода по менеджерам;
- столбцы комиссии по типу инструмента;
- матрица: подразделение → менеджер, активные клиенты, оборот, комиссия, комиссия на клиента;
- фильтры: рынок, инструмент, менеджер и сегмент.

## 4. Денежный поток

- карточки «Пополнения», «Выводы», «Чистый приток»;
- водопад: пополнения → выводы → чистый приток;
- две линии: пополнения и выводы по месяцам;
- столбцы чистого притока по клиентским сегментам.

## 5. Качество данных

- карточка «Всего ошибок качества»;
- карточка «Статус качества»;
- таблица проверок с количеством и критичностью;
- столбчатая диаграмма количества ошибок по типу;
- условное форматирование: красный — блокировка, жёлтый — предупреждение, зелёный — ошибок нет.

## 6. Подсказка по отчёту

Текстовая страница для демонстрации: бизнес-вопрос, схема модели, определения KPI, дата последнего обновления и предупреждение, что данные синтетические.

## Drill-through и подсказки

- Drill-through по клиенту: история сделок и денежных операций.
- Drill-through по менеджеру: клиентский портфель, оборот и комиссия.
- Tooltip по инструменту: рынок, тип, оборот, сделки, комиссия.

Для интервью достаточно качественно собрать первые пять страниц. Drill-through — дополнительный плюс.
